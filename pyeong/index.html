평수 계산기

제곱미터(㎡)와 평을 양방향으로 즉시 변환합니다. 한쪽에 입력하면 다른 쪽이 자동 계산됩니다.

㎡
평

1평은 몇 ㎡인가요?

1평 = 약 3.305785㎡ 이고, 1㎡ = 약 0.3025평입니다. 평에 3.305785를 곱하면 ㎡, ㎡에 0.3025를 곱하면 평이 됩니다.

아파트 평형 빠른 참조

  • 전용 59㎡ ≈ 17.8평 (흔히 24~25평형)
  • 전용 84㎡ ≈ 25.4평 (흔히 33~34평형)
  • 전용 114㎡ ≈ 34.5평 (흔히 43~45평형)

* 평형(공급면적)은 전용면적에 공용면적이 더해져 전용 평수보다 큽니다.
